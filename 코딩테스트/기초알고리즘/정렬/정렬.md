## 버블 정렬
- 가장 단순한 정렬 알고리즘으로, 1번째와 2번째, 2번째와 3번째, ... n-1번째와 n번째 값을 비교한다.
- O(n^2)의 시간 복잡도를 보여 준다.

## 선택 정렬
1. 전체 값 중 가장 작은 값을 찾음
2. 해당 값을 맨 첫번째에 배치함
3. 첫 번째 값을 제외하고 가장 작은 값을 찾아 두번째에 배치함
4. 두번째, 세번째, ..., n-1번째 값을 제외하고 가장 작은 값을 찾아 정해진 위치에 배치한다.
